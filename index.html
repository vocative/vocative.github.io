About

What is Vocative?

Vocative is a student-created Socratic Seminar-style discussion forum to facilitate classroom discussions on the modern Web.

Tools used in classrooms are often outdated, unreliable, and expensive, but this is a tool built by students, for students, for free.

What does it do?

At its core, Vocative is simply a chat application. A very pretty one.

On top of this core, though, it boasts a huge variety of features designed to make discussion flow easily, including message filtering, chat streams within the main conversation, and more.

How do I use it?

Just type in a room code above!

You can also add &admin to the end of the URL to enable administrative features.
